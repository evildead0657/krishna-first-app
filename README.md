# PRF Udyog Website

High-converting, SEO-friendly marketing and sales website for **PRF Udyog**.

## Included Sections
- Strong conversion-focused homepage hero with CTAs
- Industrial product showcase
- Trust and experience highlights
- Testimonials
- Lead generation contact form
- Floating WhatsApp chat button
- Responsive design for mobile, tablet, and desktop
- SEO essentials (meta tags, Open Graph, canonical, robots, sitemap, schema)

## GitHub Pages Deployment
1. Push this repository to GitHub.
2. Go to **Settings → Pages**.
3. Set **Source** to `Deploy from a branch`.
4. Choose branch: `main` (or your default branch), folder: `/ (root)`.
5. Save and wait for deployment.
6. Your site will be available at:
   - `https://<username>.github.io/<repository>/`

## Lead Form Setup
The contact form currently uses FormSubmit:
- Endpoint: `https://formsubmit.co/sales@prfudyog.com`

Replace `sales@prfudyog.com` with your actual lead email if needed.
